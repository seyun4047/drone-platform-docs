# MAIN-DRONE 대시보드

---
## 저장소 개요

드론 플랫폼 대시보드는
<br> 제조업체 독립형 드론 플랫폼의 프론트엔드입니다.

이 대시보드는 실시간 드론 원격 측정 및 이벤트 데이터를 시각적으로 정리하여<br> 신속한 분석을 가능하게 하고,<br> 빠르고 효과적인 재난 대응 및 수색-구조 작전을 지원합니다.

---

## 핵심 기능

- **실시간 노드 탐색**: 중앙 서버에 현재 연결된 모든 "활성" 드론을 자동으로 요청하고 추적합니다.

- **실시간 데이터 스트림**: 고주파 원격 측정 데이터(속도, 전력, GPS)와 이벤트 데이터를 수집하고 시각화합니다.

- **이벤트 인텔리전스**: "인간 감지"와 같은 중요한 경고를 관련 시각적 증거와 함께 즉시 캡처하고 표시합니다.

## 사용법

명령 센터를 몇 초 만에 가동하십시오:

```bash
# Install dependencies
npm install

# Start the dashboard in development mode
npm run dev
```
