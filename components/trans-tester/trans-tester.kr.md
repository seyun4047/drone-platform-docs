# 드론 데이터 전송 테스터

---
## 저장소 개요
이 저장소는 드론 연결, 텔레메트리, 이벤트 데이터를 시뮬레이션하여 서버의 API 동작을 검증하는 **드론 데이터 전송 테스터**를 제공합니다.
---

## 작동 방식

| Step | API Endpoint             | 설명                                           | 목적                                    | 비고        |
|------|--------------------------|------------------------------------------------|-----------------------------------------|-------------|
| 1    | `/auth/connect`          | 드론 시리얼 및 장치 이름을 전송합니다. 승인되면 인증 토큰을 받습니다. | 드론과 서버 간에 유효한 세션을 설정합니다. |             |
| 2    | `/api/telemetry`         | 토큰과 함께 일반 텔레메트리 데이터(각도, 위치)를 전송합니다. | 주기적인 드론 상태 정보를 전송합니다.     | (event=0)   |
| 3    | `/api/telemetry`         | 이벤트 플래그 및 이벤트 상세 정보(예: 사람 감지됨)를 포함한 텔레메트리를 전송합니다. | 중요한 감지 이벤트를 보고합니다.           | (event=1)   |
| 4    | `/auth/update`           | 현재 토큰을 전송하고 갱신된 토큰을 받습니다.           | 유효한 인증된 세션을 유지합니다.             |             |
| 5    | `/auth/disconnect`       | 토큰과 함께 연결 해제 요청을 전송합니다.                 | 연결을 깔끔하게 종료합니다.                   |             |

---

## 설치
필요한 종속성을 설치합니다:
```bash
pip install -r requirements.txt
```
---
## 사용법
애플리케이션을 실행합니다:
```bash
python3 main.py
```
