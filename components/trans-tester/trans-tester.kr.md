# 드론 데이터 전송 테스터

---
## 레포지토리 개요
이 레포지토리는 드론 연결, 텔레메트리 및 이벤트 데이터를 시뮬레이션하여 서버의 API 동작을 검증하는 **드론 데이터 전송 테스터**를 제공합니다.
---

## 작동 방식

| 단계 | API 엔드포인트             | 설명 | 목적 | 비고      |
|------|--------------------------|-------------|---------|-----------|
| 1 | `/auth/connect`          | 드론 시리얼 및 장치 이름을 전송합니다. 승인되면 인증 토큰을 받습니다. | 드론과 서버 간의 유효한 세션 구축 | |
| 2 | `/api/telemetry`| 토큰과 함께 일반 텔레메트리 데이터(각도, 위치)를 전송합니다. | 주기적인 드론 상태 정보 전송 | (이벤트=0) |
| 3 | `/api/telemetry` | 이벤트 플래그 및 이벤트 상세 정보(예: 사람 감지됨)와 함께 텔레메트리를 전송합니다. | 중요한 감지 이벤트 보고 | (이벤트=1) |
| 4 | `/auth/update`           | 현재 토큰을 전송하고 갱신된 토큰을 받습니다. | 유효한 인증 세션 유지 | |
| 5 | `/auth/disconnect`       | 토큰과 함께 연결 해제 요청을 전송합니다. | 연결을 깔끔하게 종료 | |
