# 드론 데이터 전송 테스터

---
## 저장소 개요
이 저장소는 서버의 API 동작을 확인하기 위해 드론 연결, 텔레메트리 및 이벤트 데이터를 시뮬레이션하는 **드론 데이터 전송 테스터**를 제공합니다.
---

## 작동 방식

| Step | API Endpoint             | Description | Purpose | etc.      |
|------|--------------------------|-------------|---------|-----------|
| 1 | `/auth/connect`          | 드론 시리얼 및 장치 이름을 전송합니다. 승인되면 인증 토큰을 받습니다. | 드론과 서버 간의 유효한 세션 설정 |
| 2 | `/api/telemetry`| 토큰과 함께 일반 텔레메트리 데이터(각도, 위치)를 전송합니다. | 주기적인 드론 상태 정보 전송 | (event=0) |
| 3 | `/api/telemetry` | 이벤트 플래그와 이벤트 상세 정보(예: 사람 감지)를 포함한 텔레메트리를 전송합니다. | 중요한 감지 이벤트 보고 | (event=1) |
| 4 | `/auth/update`           | 현재 토큰을 전송하고 새로 고쳐진 토큰을 받습니다. | 유효한 인증된 세션 유지 |
| 5 | `/auth/disconnect`       | 토큰과 함께 연결 해제 요청을 전송합니다. | 연결을 깔끔하게 종료 |

---

## 설치
필수 종속성을 설치합니다:
```bash
pip install -r requirements.txt
```
---
## 사용법
애플리케이션을 실행합니다:
```bash
python3 main.py
```
